Create a web application based on the tomjs configuration, recommended `node>=18`

**English** | [中文](./README.zh_CN.md)

## Creating a Project

- Using `npm`

```bash
npm create @tomjs/app@latest
```

- Using `yarn`

```bash
yarn create @tomjs/app
```

- Using `pnpm`

```bash
pnpm create @tomjs/app
```
