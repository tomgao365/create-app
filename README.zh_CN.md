创建基于 tomjs 配置的 web 应用，推荐 `node>=18`

[English](./README.md) | **中文**

## 创建项目

- 使用 `npm`

```bash
npm create @tomjs/app@latest
```

- 使用 `yarn`

```bash
yarn create @tomjs/app
```

- 使用 `pnpm`

```bash
pnpm create @tomjs/app
```
